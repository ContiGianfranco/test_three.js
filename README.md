# three.js_terrain
Test repository for three.js terrain representation using CDB.

## [deploy](https://contigianfranco.github.io/test_three.js/)

## Run locally

1. [Install npm](https://nodejs.org/en). npm version should be at least 8.18.0

```
$ npm -v
8.18.0
```

2. Run npm install
```
$ npm install
```

3. Run vite
```
$ npx vite
```
